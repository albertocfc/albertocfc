## 🌲 Sobre

Tecnologia a serviço do **Coritiba Foot Ball Club**. Este é meu perfil institucional e reúne os projetos criados para o clube, do controle de acesso às unidades até a documentação interna.

* 🏟️ Aplicações web enxutas, rápidas de operar e fáceis de manter
* 💚 Tudo no padrão visual oficial do clube, com o coxa-ui
* 📍 Curitiba, Paraná, Brasil

<br>

## 🎨 Destaque: coxa-ui

<div align="center">

[![npm](https://img.shields.io/npm/v/coxa-ui?style=for-the-badge&label=npm&color=006B3C&logo=npm&logoColor=white)](https://www.npmjs.com/package/coxa-ui)
[![jsDelivr](https://img.shields.io/badge/CDN-jsDelivr-00544D?style=for-the-badge&logo=jsdelivr&logoColor=white)](https://cdn.jsdelivr.net/npm/coxa-ui/)
[![GitHub](https://img.shields.io/badge/GitHub-CORITIBA%20SAF%20TECH-1A2E1F?style=for-the-badge&logo=github&logoColor=fad716)](https://github.com/CORITIBA-SAF-TECH/coxa-ui)

</div>

O **coxa-ui** é o design system oficial do Coritiba SAF: um kit de CSS e JS com as cores, os tokens e os componentes do clube, pronto para usar em qualquer aplicação. Botões, cards, tabelas, modais, toasts e dark mode, tudo sem build e sem dependências. É a base visual de todos os projetos deste perfil.

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/coxa-ui/coxa-ui.css">
<script src="https://cdn.jsdelivr.net/npm/coxa-ui/coxa-ui.js" defer></script>
```

<br>

| Projeto | Descrição | Stack |
